# Portefeuille de projets de science des données

Ce dépôt présente trois projets conçus pour démontrer mes compétences en analyse de données, visualisation, modélisation et optimisation. Ils sont particulièrement pertinents pour des postes de **Data Analyst**, **Business Analyst** ou **Chargé d'études** en France.

## 1. Prévision des ventes pour les magasins de détail
Ce projet utilise des données synthétiques pour explorer les tendances de ventes et entraîner des modèles de régression (linéaire et Random Forest) afin de prévoir les ventes quotidiennes. Il montre comment créer des variables explicatives, évaluer les modèles et visualiser les résultats.

## 2. Prédiction du churn (attrition des clients)
Ce projet génère des données artificielles pour simuler le comportement d'abonnement et met en œuvre des modèles de classification (régression logistique et Random Forest) pour identifier les clients susceptibles de quitter. Les performances sont évaluées à l'aide de métriques telles que la précision, le rappel et la matrice de confusion.

## 3. Stratégie de tarification dynamique pour les sièges d'avion
Inspiré d'une étude de cas, ce projet montre comment prédire le taux d'occupation des vols avec un modèle XGBoost et appliquer une stratégie de tarification dynamique afin de maximiser la recette. Il compare la recette d'une politique de prix fixe avec une politique dynamique basée sur les jours restants et la demande prévue.

Chaque dossier contient un `README.md` (instructions, objectifs et extensions possibles) et un script `main.py` exécutant un exemple complet. L'objectif de ce portefeuille est de montrer une approche méthodique, l'utilisation de bibliothèques Python (pandas, scikit‑learn, XGBoost) et des compétences transférables à des missions d'analyste de données en France.
